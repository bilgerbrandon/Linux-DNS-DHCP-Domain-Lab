# Configuration Files

This directory contains the configuration files used throughout the Linux Active Directory Lab.

## Included Files

### smb.conf

Samba Active Directory Domain Controller configuration.

### dhcpd.conf

DHCP server configuration used for client IP assignment.

### netplan-config.yaml

Ubuntu Server network configuration.

## Purpose

These files demonstrate:

* Active Directory configuration
* DNS integration
* DHCP services
* Linux networking
* Infrastructure management

These files are included for documentation and educational purposes and may require environment-specific modifications before deployment.
